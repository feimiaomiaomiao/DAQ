# DAQ

综合数据采集
程序简介:
ADC+DMA多通道数据采集，显示在OLED上，并通过串口1发送到串口上。

V1.1.0
+++ 新增-动态圆圈采取不同的顺时针和逆时针旋转
+++ 新增-通过定时2分开了OLED显示与ADC采集

*** 优化-把数据名称采用左对齐

--- BUG：在画圆圈时，圆圈快满的时候会造成数据刷显示停顿
=== 暂时处理-表面修复：降低显示刷新频率，待深究修复底层问题

注：程序芯片为STM32F103C8T6，所有注释仅针对此芯片，其他芯片请参考相同内容章节。

******************************************************************************

******************************************************************************
连接引脚：
C8T6               Hardware

PA1   --------    滑动变阻器1
PA2   --------    滑动变阻器2
PA9   --------    UART1_TX
PA10  --------    UART1_RX

PB8   --------    OLED_SCL
PB9   --------    OLED_SDA
